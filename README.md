# ATOM Group Cleaner 🥛🛡️

سكربت لتصفير قنوات ومجموعات تليجرام عن طريق طرد الأعضاء ومسح الرسائل بالكامل باستخدام مكتبة Telethon (MTProto).

## ⚙️ آلية عمل السكربت:
1. **الاتصال بالخادم:** يتصل السكربت بسيرفرات تليجرام كعميل بوت باستخدام الـ API ID و API Hash وتوكن البوت.
2. **طرد الأعضاء (Kick):** يقوم البوت بجلب قائمة الأعضاء وطرد كل عضو (باستثناء المشرفين) عن طريق إرسال طلب حظر مؤقت ثم فكه فوراً لتسهيل عودتهم لاحقاً. يتم تطبيق تأخير 1.0 ثانية لمنع الحظر.
3. **مسح الرسائل:** يقوم بحذف الرسائل بشكل متتابع مع تطبيق تأخير 0.5 ثانية.
4. **تخطي التقييد (Flood Wait):** يعالج السكربت قيود تليجرام للانتظار تلقائياً في حال حدوثها دون انهيار البرنامج.

## 🛠️ طريقة التشغيل:

### 📱 1. التشغيل على تيرمكس (Termux):
الصق الأمر التالي في تطبيق Termux للتشغيل المباشر:
```bash
pkg install curl -y && curl -sL https://raw.githubusercontent.com/atomxtols/atom-group-cleaner/main/setup.sh | bash
```

### 💻 2. التشغيل على الكمبيوتر (PC):
1. تثبيت مكتبة الاتصال:
```bash
pip install telethon
```
2. تشغيل السكربت:
```bash
python group_cleaner.py
```

## 📥 البيانات المطلوبة عند التشغيل:
* **API ID & API Hash:** من موقع `my.telegram.org`.
* **Bot Token:** من البوت فاذر `@BotFather`.
* **Group Username / ID:** معرف القروب المستهدف.
* **رقم العملية:** طرد الأعضاء (1), مسح الرسائل (2), تصفير شامل (3).
